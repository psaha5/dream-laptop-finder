# Dream Laptop Finder

Author: Partha Saha

## How to Run

1. Open project in IntelliJ or VS Code.
2. Ensure all .java files and laptops.txt are in the same folder.
3. Compile using Java 25.
4. Run FindLaptop.java.

## Features

- Search laptops
- Uses OOP concepts
- Uses enums
- Uses JOptionPane
- File reading/writing
- Input validation
- Registry searching
- Polymorphism and inheritance
